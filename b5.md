# BÀI 5: Tạo mã nguồn mẫu qua Prompt (Sinh code Java & Unit Test)

### 1. Nội dung Prompt thiết kế (đầy đủ Goal, Context, Constraint)
> "Bạn là một lập trình viên Java Senior chuyên về Clean Code. Hãy viết một hàm kiểm tra tính hợp lệ của mật khẩu dựa trên các quy tắc nghiệp vụ sau:
> 1. Độ dài phải từ 8 ký tự trở lên.
> 2. Phải chứa ít nhất một chữ cái viết hoa (A-Z).
> 3. Phải chứa ít nhất một chữ số (0-9).
>
> Yêu cầu đầu ra:
> - Viết bằng ngôn ngữ Java thuần, đặt tên hàm rõ ràng theo chuẩn CamelCase.
> - Code phải sạch, không thừa thãi, xử lý an toàn các trường hợp dữ liệu đầu vào bị null hoặc rỗng.
> - Viết kèm các đoạn mã Unit Test (sử dụng JUnit) bao phủ đủ các kịch bản kiểm thử: mật khẩu hợp lệ, mật khẩu quá ngắn, mật khẩu thiếu chữ hoa, mật khẩu thiếu số, và mật khẩu bị null/rỗng."

### 2. Mã nguồn Java và Unit Test do AI tạo ra

#### Đoạn code hàm kiểm tra mật khẩu (`PasswordValidator.java`)
```java
public class PasswordValidator {

    /**
     * Kiểm tra tính hợp lệ của mật khẩu theo quy tắc nghiệp vụ.
     * @param password Chuỗi mật khẩu cần kiểm tra
     * @return true nếu mật khẩu hợp lệ, ngược lại là false
     */
    public static boolean isValidPassword(String password) {
        // Ràng buộc kiểm tra an toàn: dữ liệu đầu vào null hoặc rỗng
        if (password == null || password.trim().isEmpty()) {
            return false;
        }

        // Quy tắc 1: Độ dài phải từ 8 ký tự trở lên
        if (password.length() < 8) {
            return false;
        }

        boolean hasUppercase = false;
        boolean hasDigit = false;

        // Duyệt qua từng ký tự để kiểm tra Quy tắc 2 và Quy tắc 3
        for (char ch : password.toCharArray()) {
            if (Character.isUpperCase(ch)) {
                hasUppercase = true;
            } else if (Character.isDigit(ch)) {
                hasDigit = true;
            }

            // Tối ưu: Dừng sớm nếu đã thỏa mãn cả hai điều kiện
            if (hasUppercase && hasDigit) {
                return true;
            }
        }

        return false;
    }
}
